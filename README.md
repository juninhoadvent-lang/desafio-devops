# Desafio DevOps Jr - CI/CD com GitHub Actions

Este projeto tem como objetivo automatizar o processo de build e publicação de uma imagem Docker utilizando GitHub Actions.

## O que foi feito:
1. [cite_start]**Dockerfile**: Criado para containerizar a aplicação seguindo as boas práticas[cite: 8].
2. [cite_start]**Pipeline CI**: Configurada para realizar o build e push automático para o Docker Hub sempre que houver um envio para a branch principal[cite: 21, 24].
3. [cite_start]**Segurança**: As credenciais do Docker Hub foram armazenadas de forma segura utilizando **GitHub Secrets**[cite: 30].

## Como visualizar:
- A imagem pública está disponível no meu perfil do Docker Hub.
- Os registros de execução podem ser vistos na aba **Actions** deste repositório.
